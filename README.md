## Overview
QuizMe provides an engaging platform for users to test their knowledge through interactive quizzes spanning various subjects and difficulty levels. Built with simplicity and user experience in mind, it requires no backend setup or authentication.

**Who it's for:** Students, educators, trivia enthusiasts, and anyone looking to test or expand their knowledge.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Architecture**: Client-side only (no backend required)
- **Deployment**: Static hosting compatible

## Features
- **Multi-Topic Quizzes**: Wide range of subjects to choose from
- **Difficulty Levels**: Multiple difficulty settings to match your skill level
- **Interactive UI**: Clean, user-friendly interface for seamless quiz-taking
- **No Registration**: Start taking quizzes immediately
- **Lightweight**: Pure vanilla JavaScript - no framework dependencies

## Installation

1. **Clone the repository**
```bash
   git clone https://github.com/Marvellousz/QuizMe.git
   cd QuizMe
```

2. **Launch the application**
```bash
   # Simply open the index.html file in your browser
   # Or use a local server (recommended):
   
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (with npx)
   npx serve
```

3. **Access the app**
   
   Visit `http://localhost:8000` (or open `index.html` directly)

## Usage

### Taking a Quiz

1. **Open the application** in your web browser
2. **Select a topic** from the available categories
3. **Choose difficulty level** that matches your knowledge
4. **Answer questions** interactively
5. **View your results** at the end of the quiz

### Customization

The quiz data and styling can be customized by editing:
- Questions and answers in the JavaScript files
- Visual appearance in the CSS files
- Layout structure in the HTML files
